RMSTpowerBoost Pipeline

Use the workflow in order. Each of Steps 1-3 has a Confirm button (check mark) and a Reset button (round arrow). The standalone Workflow Reset is available at all times.

4-Step Flow

The next step appears only after confirming the current one. After confirmation, the previous step UI is hidden.

  1. Step 1. Data Upload or Generate: choose the source mode.
  2. Step 2. Generation or MICE: configure generation (if Generate mode) or run cleaning/MICE (if Upload mode).
  3. Step 3. Model and Mapping: select model first, then map required columns.
  4. Step 4. Analysis: set Full Truncation, calculation method, target inputs, and run.

Step 1. Data Upload or Generate

  • A) Generate Data: open the generation pathway.
  • B) Upload Pilot Data: upload .csv, .txt, .tsv, .rds, or .RData.
  • Use ✓ Confirm Step 1 to lock your choice for the current pass.
  • Use ↻ Reset Step 1 to return to Upload mode defaults.

Step 2. Generation or MICE

Generate Path

  • Define covariates and event-time settings.
  • Generate data and review validation cues.
  • Use ✓ Confirm Step 2 and ↻ Reset Step 2.

Upload/MICE Path

  • Profile missingness and apply dropping/MICE options.
  • Confirm when missingness is resolved for analysis.
  • Use ✓ Confirm Step 2 and ↻ Reset Step 2.

Step 3. Model and Mapping

  • Model Selection comes first across the full row.
  • Column mapping is next: time, status, arm, and optional strata.
  • Use ✓ Confirm Step 3 to unlock the run controls.
  • Use ↻ Reset Step 3 to restore default model + auto-mapping.

Step 4. Analysis

  • Set Full Truncation, calculation method, alpha, and target inputs.
  • Run analysis and inspect Plot Output, Summary, and Run Log.
  • You can modify Step 4 inputs repeatedly and rerun without returning to Steps 1-3.

Reset Behavior

  • ↻ Reset Workflow is standalone and returns the app to Step 1 from any stage.
  • Step-level reset buttons only reset the current stage.
